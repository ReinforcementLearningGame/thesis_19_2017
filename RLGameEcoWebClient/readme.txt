Web client

Εδώ υπάρχει ο κώδικας για τον Web client.
Στη γραμμή 5 του αρχείου .\script\app.js βάζουμε τη διεύθυνση του server του οικοσυστήματος.
Μπορούμε να δοκιμάσουμε τον client ανοίγοντας το αρχείο index.html σε κάποιον browser ή τοποθετώντας τα αρχεία σε κάποιον web server.
